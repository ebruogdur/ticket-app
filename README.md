# ticket-app
Ticket Application Demo 

Uygulamada kullanılan teknolojiler:
• JAVA 8 
• Maven 
• Spring Boot
• Hibernate
• Postgresql
• Restful Web Service 
• Junit Test 

Projenin dokümantasyonu swagger ile yapılmıştır . Proje ayağa kalktıktan sonra "http://localhost:8080/swagger-ui.html" bu linkten erişebilirsiniz .
